js总结笔记

学习中所做的笔记，希望能帮到初学者。

javascript

输出内容（document.write）
1.输出内容内容用“”括起来，直接输出“”内的内容
Document.write(“I love javascript”);
2.通过变量，输出内容
Document.write(mynum); //直接输变量名，不用“”
3.输出多项内容，中间用+连接
4.输出HTML标签，并起作用，标签用“”括起来

警告 alert window.alert
用法同上，变量不加“” 多项内容用+连接

确认 （confirm消息对话框）
Confirmstr(str); str:在对话框中要显示的文本 返回值：boolean值

提问 (prompt 消息对话框)
Prompt(str1,str2); str1:要显示在消息对话框中的内容，不可修改
Str2：文本框中的内容，可以修改（注意中间是逗号隔开，而不是加号 str2一般是函数名）

打开新窗口 (window.open)

关闭窗口 window.close

控制Dom流
1.通过元素获取元素 document.getElementById(“id”)
2.innerHTML属性 object.innerHTML object是获取的元素对象，如通过document.getElementById(“id”)获取的元素
3.改变HTML样式 object.style.property=new style; object是获取的元素对象
4.显示和隐藏 display属性 object.style.display = value 其值 none为隐藏；block为将此元素设置为块级元素，即显示
5.控制类名 object.className = classname 作用：获取元素的class属性；为网页内的某个元素指定一个class样式来更改该元素的外观。
Object为获取的元素对象 classname为样式表中定义的css样式类名

创建数组 var myarray = new Array(); var myarray保存数组的变量 new Array创建一个新的空数组，还可指定长度

事件 主要事件罗列
1.onclick 鼠标点击事件
2.Onmouseover 鼠标经过事件
3.Onmouseout 鼠标移出事件
4.Onchange 文本框内容改变事件
5.Onselect 文本框内容被选中事件
6.Onfocus 光标聚集事件
7.Onblur 光标移开事件
8.Onload网页导入事件(一般写在body标签内)
9.Onunload 关闭网页事件
