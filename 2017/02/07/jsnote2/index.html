javascript

定义时间对象 var Udate=new Date() ;
Udate 为时间对象，并且已有初始值；初始值为当前时间。即当前电脑系统时间。

1.返回/设置年份 get/setFullYear() 用四位数表示
2.返回星期 getDay()
4.getMonth 返回月份
5.getDate 返回日期

1.字符串对象
字符串对象变为大写 字符串对象.toUpperCase();
字符串对象又大写变为小写 字符串对象.toLowerCase();
2.返回指定位置的字符 stringObject.charAt(index)
Index表示字符串中某个位置的数字，即字符在字符串中的下标，注字符串中第一个字符的下标为0，最后一个字符的下标为字符串长度减1。如果参数不在0与string.length-1之间，将返回一个空字符串。
3.返回指定的字符串首次出现的位置 stringObject.indexOf(substring,startpos)
Substring 规定要检索的字符串值 startpos可选的整数参数
4.字符串分割split() 将字符串分割为字符串数组，并返回此数组
stringObject.split(separator,limit)
Separator 从该参数指定的地方分割 stringObject
Limit 可选参数，分割的次数
5.提取字符串substring() 用于提取字符串中介于两个指定下标之间的字符
StringObject.substring(startPos,stopPos)
starPos 必需，一个非负的整数，开始位置
stopPos 可选，一个非负的整数，结束位置
6.提取指定数目的字符 substr()
stringObject.substr(starPos,length) starPos:数值，起始位置 length可选 若不填，默认到字符串结束

Math 对象 ：区别，math对象为固有对象，无需创建它，直接把math作为对象使用就可以调用其所有的属性和方法
1.向上取整 Math.ceil(x) x数值，必需
2.向下取整 Math.floor(x) x数值，必需
3.四舍五入 Math.round(x) x必需，数字
4.随机数 Math.random() 可返回介于0~1之间的一个随机数
若想返回0~10，在（）后10，但需要用（）括起来
例 ((Math.random())10);

数组方法：
1.数组连接
arrayObject.concat(array1,array2,…arrayN)
例要将两个数组连接 var myarr3= myarr1.concat(myarr2);
2.指定分隔符连接数组元素 arrayObject.join(分隔符)
3.颠倒数组元素顺序 arrayObject.reverse()
4.选定元素 arrayObject.slice(start,end)
start规定从何处开始选取，正数，从头往后，负数，倒着数
End规定从何处结束，正数负数同上
5.数组排序 arrayObject.sort(方法函数)
方法函数：可选，规定排序顺序，必须是函数

javascript 计时器
setTimeout() 指定的延迟时间之后来执行代码
clearTimeout（） 取消setTimeout（）设置
setInterval（） 每隔指定的时间执行代码
clearInterval（） 取消setInterval（）设置

控制dom流
1.返回带有指定名称的节点对象的集合
document.getElementsByName(name)
2.返回带有指定标签名的节点对象的集合，返回元素的顺序是他们在文档中的顺序 document.getElementsByTagName(Tagname)
Tagname是标签的名称，如p，a，img等标签
3.※区别 getElementById ,getElementsByName ,getElementsByTagName
ID是一个人的身份证号码，是唯一的；name是他的名字，可以重复；tagname 可看做某类，获取相同类的集合
4.getAttribute（）方法 通过元素节点的属性名称获取属性的值
elementNode.getAttribute(name) elementNode：使用getElementById（），getElementsByTagName（）等方法，获取到的元素节点,，定义的变量值。 Name：要想查询的元素节点的属性名称。
☆☆☆ 变量不唯一时，用数组表示 变量[i]
5.setAttribute() 方法 ：增加一个指定名称和值的新属性，或者把一个现有的属性设定为指定的值 elementNode.setAttribute(name,value) name:要设置的属性名 value：要设置的属性值
6.其他属性 1.nodeName ：节点的名称 用法elementNode.nodeName
2.nodeValue:节点的值 ，用法同上
3.nodeType：节点的类型，用法同上
7.访问子节点childNodes
语法 ：elementNode.childNodes 访问选定元素节点下的所有子节点的列表，返回的值可以看做是一个数组，它具有length属性
8.访问子节点的第一或最后一项
语法：node.firstChild 或 node.lastChild
9.访问父节点
语法：elementnode.parentNode 用法同上
10.访问兄弟节点 语法 ： nodeObject.nextSibling 返回某个节点之后紧跟的节点（处于同一树层级中）
语法：nodeObject.previousSibling 返回某个节点之前紧跟的节点
11.插入节点appendChild（） ：在指定节点的最后一个子节点列表之后添加一个新的子节点
语法 appendChild(newnode) newnode:指定追加的节点
12.插入节点insertBefore（） 可在已有的节点之前插入一个新的子节点
语法：insertBefore(newnode,node) newnode :要插入的新节点 node 指定此节点前插入新节点
13.删除节点 removeChild（）
从子节点列表中删除某个节点，如删除成功，此方法可返回被删除的节点，如失败，则返回null nodeObject.removeChild(node) node:必需，指定要删除的节点
14.替换元素节点 replaceChild（） 实现子节点（对象）的替换，返回被替换对象的引用。 语法：node.replaceChild(newnode,oldnew) newnode:必需，用于替换oldnew的对象， oldnew：必需，被newnode替换的对象
15.创建元素节点 createElement 可创建元素节点，此方法可返回一个element对象
语法：document.createElement(tagName)
tagName: 字符串值，这个字符串用来指明创建元素的类型
16.创建文本节点createTextNode 创建新的文本节点，返回新创建的text节点
语法：document.createTextNode(data);
Data:字符串值，可规定此节点的文本
17.获取网页尺寸offsetHeight
offsetWidth和offsetHeight，获取网页内容宽度和高度（包括滚动条等边线，会随窗口的显示大小改变）
offsetHeight = clientHeight+滚动条+边框
